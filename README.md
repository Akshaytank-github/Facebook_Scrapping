# Facebook_Scrapping
A facebook scrapper without using facebook-api which uses a library named **facebook_scrapper**

## What you can fetch

* post_id
* post_text
* likes
* comments
* shares
* time
* image_url
* post_url

**You can fetch all these details from an public facebook page or public facebook group.
You can fetch upto 720 pages ( this may change from computer to computer based on internet speed and computing power ).**
